# Game Development

## Description

This is a project game made by students of CITM in 2023 as a part of a subject.
## Key Features

 - F3 moves the player to the start
 - F5 save game
 - F6 load game
 - F9 debug mode
 - F10 god mode
 - F11 FPS cap
 
## Controls

 - W, A, S, D to move
 - SPACE to jump
 - F to attack
